## Color Palette

The default theme uses a dark palette:

- **Primary**: #e6a817 (Gold)
- **Background**: #0d1117
- **Surface**: #1a202c
- **Border**: #2d3748
- **Text Primary**: #e2e8f0
- **Text Secondary**: #a0aec0
- **Text Muted**: #718096

## Column Colors

Each board column has its own accent color, configured in `board.md` frontmatter.

## Typography

System font stack for UI, monospace for code and ticket IDs.
